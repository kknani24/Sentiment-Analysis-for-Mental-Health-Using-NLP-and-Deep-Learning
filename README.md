# Sentiment Analysis for Mental Health Using NLP and Deep Learning

## Overview

This project applies Natural Language Processing (NLP) and Deep Learning techniques to analyze and classify sentiment in mental health-related text data. The primary goal is to classify the sentiment expressed in textual data into various mental health status categories using a Convolutional Neural Network (CNN).
